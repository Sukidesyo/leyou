# leyouMall
分布式商城项目
## 项目技术
SpringBoot    
SpringCloud       
ElasticSearch       
RabbitMQ         
Redis     
Ngnix   
## 项目分布式分支
- leyou-auth      远程客户端调用
- leyou-cart      购物车
- leyou-common    工具类
- leyou-gateway   网关
- leyou-ietm      业务逻辑代码
- leyou-registry  服务注册中心
- leyou-search    搜索
- leyou-sms       邮件发送 
- leyou-upload    文件上传
- leyou-user      用户注册等逻辑
- leyou-web-page  网页渲染
